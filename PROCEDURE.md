# 作業手順書

**作業名:** Google フォーム GAS 一括書き換え
**対象ツール:** `kondoyuri-design/google-form-gas-updater`
**作成日:** 2026年2月22日
**改訂:** 1.0

---

## 概要

Google Drive の指定フォルダ内にある全 Google フォームのスクリプトを、同一の内容に一括書き換えする手順を記す。

**作業の全体像:**

```
フェーズ1: 初回セットアップ（初回のみ）
  └─ STEP 1〜4: API 有効化・GAS プロジェクト作成

フェーズ2: 書き換え内容の設定
  └─ STEP 5〜6: フォルダ ID・新スクリプトの設定

フェーズ3: ドライランで対象確認（必須）
  └─ STEP 7〜8: dryRun 実行・ログ確認

フェーズ4: 本番実行
  └─ STEP 9〜10: 書き換え実行・完了確認
```

> **注意:** 書き換えは元に戻せません。必ずフェーズ3のドライランで対象を確認してから本番実行してください。

---

## フェーズ1: 初回セットアップ（初回のみ）

### STEP 1: Apps Script API を有効化する

1. ブラウザで以下の URL を開く
   ```
   https://console.cloud.google.com/apis/library/script.googleapis.com
   ```
2. 対象の Google アカウントでログインしていることを確認する
3. 「**有効にする**」ボタンをクリックする

   > 「管理」と表示されている場合はすでに有効。次の STEP へ進む。

---

### STEP 2: GAS スタンドアロンプロジェクトを作成する

1. ブラウザで以下の URL を開く
   ```
   https://script.google.com
   ```
2. 「**新しいプロジェクト**」をクリックする
3. 左上のプロジェクト名（「無題のプロジェクト」）をクリックして名前を変更する
   - 例: `フォームGAS一括書き換えツール`
4. エディタに表示されている `Code.gs` の内容をすべて削除する
5. 本リポジトリの `Code.gs` の内容をコピーして貼り付ける
6. `Ctrl + S` で保存する

---

### STEP 3: Apps Script API をサービスとして追加する

1. GAS エディタ左メニューの「**サービス**」横にある「**+**」をクリックする
2. 一覧から「**Apps Script API**」を選択する
3. バージョンが「**v1**」になっていることを確認する
4. 「**追加**」をクリックする

   > 左メニューの「サービス」の下に `Script` が表示されれば成功。

---

### STEP 4: マニフェストファイル（appsscript.json）を設定する

1. GAS エディタ左上の歯車アイコン「**プロジェクトの設定**」をクリックする
2. 「**「appsscript.json」マニフェスト ファイルをエディタで表示する**」にチェックを入れる
3. エディタ左のファイル一覧に `appsscript.json` が追加されたことを確認する
4. `appsscript.json` をクリックして開く
5. 内容をすべて削除し、本リポジトリの `appsscript.json` の内容を貼り付ける
   ```json
   {
     "timeZone": "Asia/Tokyo",
     "dependencies": {},
     "exceptionLogging": "STACKDRIVER",
     "runtimeVersion": "V8",
     "oauthScopes": [
       "https://www.googleapis.com/auth/drive.readonly",
       "https://www.googleapis.com/auth/script.projects"
     ]
   }
   ```
6. `Ctrl + S` で保存する

---

## フェーズ2: 書き換え内容の設定

### STEP 5: フォルダ ID を確認する

1. Google Drive で対象フォルダを開く
2. ブラウザのアドレスバーから URL を確認する
   ```
   https://drive.google.com/drive/folders/XXXXXXXXXXXXXXXX
                                           ↑ この部分がフォルダ ID
   ```
3. フォルダ ID をコピーしておく

---

### STEP 6: Code.gs の設定を編集する

1. GAS エディタで `Code.gs` を開く
2. ファイル上部の設定セクションを編集する

   **① FOLDER_ID を設定する**
   ```javascript
   const FOLDER_ID = 'STEP5 でコピーしたフォルダID';
   ```

   **② DRY_RUN が true になっていることを確認する**
   ```javascript
   const DRY_RUN = true;  // ← true のままにしておく
   ```

   **③ NEW_SCRIPT_CODE に新しいスクリプトを貼り付ける**
   ```javascript
   const NEW_SCRIPT_CODE = `
   // ここに書き換え後のスクリプトを貼り付ける
   function onFormSubmit(e) {
     // ...
   }
   `;
   ```

3. `Ctrl + S` で保存する

---

## フェーズ3: ドライランで対象確認（必須）

### STEP 7: dryRun を実行する

1. GAS エディタ上部のドロップダウン（関数選択）から「**dryRun**」を選択する
2. 実行ボタン（**▶**）をクリックする
3. 初回実行時は権限の承認ダイアログが表示される
   1. 「**権限を確認**」をクリックする
   2. 対象の Google アカウントを選択する
   3. 「**詳細**」→「**〜（安全ではないページ）に移動**」をクリックする
   4. 「**許可**」をクリックする

---

### STEP 8: 実行ログを確認する

エディタ下部の「**実行ログ**」に以下のような内容が表示されることを確認する。

```
=== ドライラン: 対象フォームとスクリプトを確認します ===

フォルダ検索中... (ID: XXXXXXXX)
フォームを N 件発見

スクリプトプロジェクトを検索中...
  ページ 1 検索済み（累計 N 件一致）
対象スクリプトを N 件特定

📋 対象: "フォーム名"
   フォームID : AAAAAAAAAA
   スクリプトID: BBBBBBBBBB
   タイトル  : フォーム名

=== ドライラン完了 ===
更新予定: N 件 / スクリプトなし: 0 件
```

**確認ポイント:**

| 確認事項 | 問題がある場合の対処 |
|----------|----------------------|
| フォームの件数が正しいか | `FOLDER_ID` が正しいか確認する |
| 全フォームが「📋 対象」に表示されているか | 「スクリプトなし」のフォームは STEP 8a を実施する |
| エラーが表示されていないか | 「トラブルシューティング」を参照する |

**STEP 8a: スクリプトが存在しないフォームへの対応**

「スクリプトなし」と報告されたフォームは、スクリプトエディタが一度も開かれていないため、スクリプトプロジェクトが存在しない。以下の手順でスクリプトを生成してから dryRun を再実行する。

1. 該当の Google フォームを開く
2. 右上メニュー「**︙**」→「**スクリプト エディタ**」をクリックする
3. GAS エディタが開いたことを確認して閉じる（内容は変更不要）
4. STEP 7 に戻り dryRun を再実行する

---

## フェーズ4: 本番実行

### STEP 9: 本番実行する

> **前提:** STEP 8 のドライランで対象が正しいことを確認済みであること。

1. `Code.gs` の `DRY_RUN` を `false` に変更する
   ```javascript
   const DRY_RUN = false;  // ← false に変更
   ```
2. `Ctrl + S` で保存する
3. 上部ドロップダウンから「**updateAllFormScripts**」を選択する
4. 実行ボタン（**▶**）をクリックする
5. 確認ダイアログが表示されたら内容を確認し「**OK**」をクリックする

---

### STEP 10: 完了を確認する

**① 実行ログを確認する**

```
✅ 更新完了: "フォーム名" (scriptId: XXXXXXXX)
✅ 更新完了: "フォーム名" (scriptId: XXXXXXXX)
...
=== 完了 ===
成功: N 件 / エラー: 0 件 / スクリプトなし: 0 件
```

**② 任意のフォームで内容を目視確認する**

1. 書き換えた Google フォームのうち1件を開く
2. 右上メニュー「**︙**」→「**スクリプト エディタ**」をクリックする
3. `Code.gs` の内容が `NEW_SCRIPT_CODE` で設定した内容になっていることを確認する

---

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| `403 Forbidden` が表示される | Apps Script API が未有効、またはサービス未追加 | STEP 1 と STEP 3 を再確認する |
| フォームが 0 件と表示される | `FOLDER_ID` が誤り、またはフォームがサブフォルダにある | STEP 5 を再確認する |
| 「スクリプトなし」と報告される | フォームにスクリプトが存在しない | STEP 8a を実施する |
| 権限エラーが表示される | OAuth の承認が不完全 | GAS エディタから再度実行して承認し直す |
| 実行がタイムアウトする | フォーム数が多く 6 分を超えた | フォルダを複数に分けて分割実行する |
